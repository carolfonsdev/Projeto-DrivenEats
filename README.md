# 🍽️ DrivenEats - Driven Bootcamp 🚀

## Description 📝

DrivenEats is a project developed as part of the Driven course, simulating a food delivery ordering service. The goal was to practice HTML, CSS, JavaScript and responsiveness, following a layout provided in Figma and ensuring a smooth experience on mobile devices.

> 🌟 **First project built with JavaScript!**

🔗 **Live Demo:** ([https://driven-eats-nu.vercel.app/](https://driven-eats-nu.vercel.app/))

---

## Tech Stack 🛠️

- HTML5
- CSS3 (with box-shadow and overflow-x: scroll)
- JavaScript
- Google Fonts: Righteous & Roboto
- Git & GitHub

---

## Features 🚀

✅ Mobile layout with fixed top bar and order button  
✅ Single selection per category (Main Dish, Drink, Dessert)  
✅ "Close order" button enabled only after all categories are selected  
✅ Order confirmation screen with item summary and total price  
✅ Automatic order submission via WhatsApp Web
